# 礼金簿 Android App 

## 开发工具
Android Studio 3.1.3 

## 功能需求

- 参考iOS版本：https://itunes.apple.com/cn/app/li-jin-bu-hong-bao-zhang-ben/id987247088?mt=8

### 功能列表：
```
1.  亲友团
2.  流水账
3.  项目表
4.  发现 
    - 导出数据
5.  密码    
6. 数据保存
7. 数据云同步
```

### 要求
```
1，要求数据不能丢失
2，要求适配主流的设备，尤其是Pad
3，要求高覆盖率Unit Test
4，建议使用MVP模式

```
